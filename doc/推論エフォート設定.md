# 推論エフォート設定（Reasoning Effort）

o3・o4-mini などの推論モデルを使う際に、**推論の深さを Low / Medium / High から選択できる機能**（x-High は一部モデル・環境のみ対応）。
モデルピッカーのサブメニューから選択できる（x-High は settings.json で `"xhigh"` と手動設定が必要な場合あり）。

- 複雑なアーキテクチャ設計・デバッグ → High / x-High
- 通常の実装・コードレビュー → Medium
- フォーマット整形・リネーム・単純な補完 → Low（または推論モデル自体を使わない）

効果★☆☆
