## GitHubリポジトリ設定

GitHub のリポジトリ設定（Settings > Copilot > Content exclusion）でパスパターンを指定すると、Copilot の参照対象からファイルを除外できる。
Copilot 拡張機能が GitHub にサインインした状態でリポジトリを開いたときに自動で読みに行きます。
ただしAgent Mode(いつものAIチャット欄で調査やソース編集する機能) では**現時点でこの設定が機能しない**ので効果は限定的です。
