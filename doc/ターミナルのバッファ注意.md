## @terminal はターミナルの履歴を丸ごと読むことがある

Copilot Chat で `@terminal` を使うと、ターミナルに表示されている内容を Copilot が読んで回答してくれる。ふだんは**最後に実行したコマンドの結果だけ**を読む。なので普通に使っている分にはそんなにトークンを食わない。

ただし`npm run dev`などで**ターミナルに出力がたくさん溜まっている状態**だと話が変わる。
Copilot は**バッファ（ターミナルに残っている全行）を最初から読みに行く**。
VS Code のデフォルトは 1000 行だが、スクロールバックを 10 万行に設定していた人が実際にこれを踏んで VS Code ごとクラッシュした報告がある。Microsoft は「仕様通りの動作」として対応を終了している。

対策として`clear`コマンドでターミナルの内容をリセットする
