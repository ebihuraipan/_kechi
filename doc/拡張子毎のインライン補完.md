# インライン補完の拡張子別無効化

`github.copilot.enable` に言語IDをキーとしたオブジェクトを指定する（VSCode設定）。

## パターン A: デフォルト有効・特定のみ無効

```json
{
  "github.copilot.enable": {
    "*": true,
    "plaintext": false,
    "markdown": false,
    "yaml": false,
    "json": false,
  }
}
```

## パターン B: デフォルト無効・特定のみ有効（最節約）

```json
{
  "github.copilot.enable": {
    "*": false,
    "javascript": true,
    "typescript": true,
    "python": true,
  }
}
```

## 主な言語ID

| 拡張子 | 言語ID |
|---|---|
| `.js` | `javascript` |
| `.ts` | `typescript` |
| `.py` | `python` |
| `.md` | `markdown` |
| `.yaml` `.yml` | `yaml` |
| `.env` | `dotenv` |
| `.sh` | `shellscript` |

言語IDはVSCodeステータスバー右下の言語名をクリック→「言語IDをコピー」で確認できる。

## 注意

- インライン補完のみ対象。チャット・エージェントには効かない。
- `.copilotignore`（ファイル自体をコンテキストから除外）とは別設定。
- 設定変更後は即時反映（再起動不要）。
