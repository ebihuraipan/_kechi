# チャットツールの絞り込み

エージェントモードで有効にしているツールは、使用しなくてもその定義がリクエストのコンテキストに乗り続ける。不要なツールをオフにするだけで毎リクエストのトークンを削減できる。

AIチャット欄の**コネクタのような横線２本が縦に並んでるアイコン**から
ツールのON/OFFを切り替えられる。

| ツール | 定義数 | 推奨 | 理由 |
|--------|--------|------|------|
| agent | 1 | **オフ** | サブエージェント委任は不要 |
| browser | 10 | **オフ** | ブラウザ自動化は不要 |
| execute | 8 | オン | ターミナル・タスク実行の中核 |
| new | 0 | **オフ** | スキャフォールドは稀 |
| read | 3 | オン | タスク・ターミナル出力の読み取り |
| todo | 0 | **オフ** | ToDo 管理が不要なら除外 |
| vscode | 2 | オン | VS Code 基本機能 |

`browser`(10) + `agent`(1) をオフにするだけで **11個分のツール定義**がコンテキストから除外される。


効果★★☆
